## Installation 
```sh
npm add br-doc-validator
# or
yarn add br-doc-validator
```

## Usage

```javascript
const validator = require('br-doc-validator');
const document = '00000000353';
const result = validator.documentValidator(document);

if(result) {
    console.log('valid');
} else {
    console.log('invalid');
}
```
```sh
Output should be 'valid'
```
